HTML Notes

  • <!DOCTYPE> - declares the type of document being used, in this case HTML5

  • <html> - represents the root element of an HTML page

  • <head> - contains meta information about the document, such as the title and character encoding

  • <body> - contains the main content of the HTML document

  • <header> - represents a container for introductory content or a set of navigational links
